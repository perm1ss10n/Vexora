# Vexora Platform

**Vexora** — аппаратно-программная IoT-платформа для сбора, доставки, хранения и анализа телеметрических данных с распределённых устройств через комбинированные каналы связи (Wi-Fi / GSM).

Платформа спроектирована для надёжной работы в условиях нестабильного или отсутствующего интернета и ориентирована на коммерческое масштабирование.

## Компоненты платформы
- **Vexora Box** — аппаратный бокс (ESP32 + датчики + GSM)
- **Vexora Agent** — прошивка устройства
- **Vexora Cloud** — backend, MQTT, хранилище данных
- **Vexora App** — кроссплатформенное desktop-приложение (Windows / macOS / Linux)

## Ключевые возможности
- автоматический fallback Wi-Fi → GSM
- управление устройствами через MQTT
- централизованная конфигурация и OTA
- хранение телеметрии в time-series БД
- визуализация и экспорт данных
- multi-tenant архитектура
